报表API

Apptalking 控制台支持任意指标导出，可以非常方便的和第三方应用结合，使用HTTP协议，支持xml和json格式。

API 参数

URL地址：{应用安装路径}/controller/rest/applications/{系统名称}/metric-data
参数列表如下：

  • output[json , xml]
    性能指标导出的格式，json或者xml格式，默认使用xml格式。
  • rollup[true, false]
    是否汇总性能指标数据、默认true
  • use-cache-data[true , false]
    是否使用缓存数据，默认false
  • metric-path
    性能指标路径，可以在性能指标库中查询，复制获取。
  • time-range-type[BEFORE_NOW , BETWEEN_TIMES]
    时间戳类型，BEFORE_NOW 在当前时间以前，BETWEEN_TIMES 指定时间的范围。
  • duration-in-mins
    时间长度，分钟数，配合BEFORE_NOW 类型使用。
  • start-time
    开始时间，配合BETWEEN_TIMES类型使用
  • end-time
    结束时间，配合BETWEEN_TIMES类型使用

从指标库中复制指标路径

点击复制完整路径复制Metric Path。
点击复制Restful Url 复制 完整的API url。
