Apptalking Server 安装

Apptalking Server 是数据中心，用于数据收集和报表呈现。

1、Apptalking Server 下载安装

  • 下载
    在Apptalking官网下载 Apptalking Server
  • 安装
    Apptalking Server 解压到安装目录就可以使用，无需其他安装操作。

2、安装数据库

  • 安装 Mysql
    Apptalking Server 使用的是Mysql数据库，数据库下载安装参考 Mysql 官网
  • 导入数据库表信息
    使用 apptalking 名称创建数据库， 导入Apptalking Server 安装包中 db/apptalking.sql 数据库文件。
  • 数据库IP、密码修改
    修改 ApptalkingServer\glassfish3\glassfish\domains\domain1\config\domain.xml 配置文件

3、启动、停止服务

启动脚在ApptalkingServer\bin 文件夹下。

  • 启动服务
    • window
      apptalking.bat start
    • liunx
      apptalking start
  • 停止服务
    • window
      apptalking.bat stop
    • liunx
      apptalking stop
